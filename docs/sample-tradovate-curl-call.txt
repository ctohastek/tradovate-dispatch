# Sample Tradovate API Authentication Call
# This is the /auth/accesstokenrequest endpoint - NOT a trading command

## Authentication Flow
1. Call this endpoint to get an access token
2. Use that token (Bearer <accessToken>) for all subsequent API calls
3. Access tokens expire after 90 minutes - call /auth/renewaccesstoken to extend

## Parameter Reference
 - name: Account name used for authentication (from TRADOVATE_ACCOUNT_NAME in .env)
 - password: Password set during API key generation (from TRADOVATE_ACCOUNT_PASS in .env)
 - appId: Tradovate API key nickname (from agents.yaml appId field)
 - appVersion: Version of your app (e.g., "1.0")
 - deviceId: Unique device identifier (from TRADOVATE_DEVICE_ID in .env, same for all agents)
 - cid: Client ID issued with API key (from TRADOVATE_CLIENT_ID_<AGENT> in .env)
 - sec: API secret issued with API key (from TRADOVATE_API_KEY_<AGENT> in .env)

## Example curl call:

curl -X POST "https://demo.tradovateapi.com/v1/auth/accesstokenrequest" \
 -H "accept: application/json" \
 -H "Content-Type: application/json" \
 -d "{ \
   \"name\": \"mini01\", \
   \"password\": \"password-set-during-api-key-generation\", \
   \"appId\": \"mini01\", \
   \"appVersion\": \"1.0\", \
   \"deviceId\": \"device-id-from-TRADOVATE_DEVICE_ID-env\", \
   \"cid\": \"12026\", \
   \"sec\": \"your-api-secret-from-key-generation\" \
 }"

## Response (Success):
{
  "accessToken": "ag_8a97la5-T6PqMfUhu-NReEUxW4cFq_dxw...",
  "expirationTime": "2026-05-14T15:45:00Z",
  "userId": 12345,
  "name": "mini01"
}

## Then use accessToken for trading calls:

curl -X GET "https://demo.tradovateapi.com/v1/account/list" \
 -H "Authorization: Bearer ag_8a97la5-T6PqMfUhu-NReEUxW4cFq_dxw..." \
 -H "Content-Type: application/json"
